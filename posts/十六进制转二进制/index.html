北邮VC++实验题 将十六进制数转换为二进制数

题目描述

将十六进制数转换为二进制数。

输入格式

输入一个16进制数。

输出格式：

输出二进制数。

样例

样例输入

23

样例输出

100011

解答

#include <iostream>
#include <cstring>
#include <bitset>
//十六进制转二进制
using namespace std;
int main()
{
    int n,i=0,s[10000];
    scanf("%x",&n);
    while(n!=0)
    {
        s[i]=n%2;
        //printf("%d\n",s[i]);
        n/=2;
        i++;
    }
    for(int j=i-1;j>=0;j--)
    {
        printf("%d",s[j]);
    }
    if(i==0)
    {
        printf("0");
    }
}
