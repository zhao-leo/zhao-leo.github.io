北邮VC++实验题 判别水果颜色

题目描述

现在有一堆水果，苹果红色，香蕉黄色，西瓜绿色，其余紫色。 当输入一种水果名字，要输入其对应的颜色。

输入格式

输入水果名字，如：苹果。

输出格式

输出颜色，如：红色。

样例

输入样例

苹果

输出样例

红色

解答

#include <iostream>
#include <cstring>
//判断水果颜色
using namespace std;
int main()
{
    string s;
    cin>>s;
    if(s=="苹果")cout<<"红色";
    else if(s=="香蕉")cout<<"黄色";
    else if(s=="西瓜")cout<<"绿色";
    else cout<<"紫色";
}
