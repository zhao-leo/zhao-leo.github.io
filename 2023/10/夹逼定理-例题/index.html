一道数列极限

题目如下：

求下列数列极限：

lim⁡n→∞(1n3+1+4n3+2+⋯+n2n3+n)\lim_{n \to \infty} \left ( \frac{1}{n^{3}+1 }+ \frac{4}{n^{3}+2 }+\dots+\frac{n^{2} }{n^{3}+n } \right ) n→∞lim​(n3+11​+n3+24​+⋯+n3+nn2​)

解答：

第一步：

lim⁡n→∞(1n3+1+4n3+2+⋯+n2n3+n)>lim⁡n→∞(1n3+n+4n3+n+⋯+n2n3+n)=lim⁡n→∞(16×n×(n+1)×(2n+1)n3+n)=13\lim_{n \to \infty} \left ( \frac{1}{n^{3}+1 }+ \frac{4}{n^{3}+2 }+\dots+\frac{n^{2} }{n^{3}+n } \right )>\\ \lim_{n \to \infty} \left ( \frac{1}{n^{3}+n }+ \frac{4}{n^{3}+n }+\dots+\frac{n^{2} }{n^{3}+n } \right )=\\ \lim_{n \to \infty} \left ( \frac{\frac{1}{6}\times n\times \left ( n+1 \right ) \times \left ( 2n+1 \right ) }{n^{3}+n } \right )=\frac{1}{3}n→∞lim​(n3+11​+n3+24​+⋯+n3+nn2​)>n→∞lim​(n3+n1​+n3+n4​+⋯+n3+nn2​)=n→∞lim​(n3+n61​×n×(n+1)×(2n+1)​)=31​

第二步：

lim⁡n→∞(1n3+1+4n3+2+⋯+n2n3+n)<lim⁡n→∞(1n3+1+4n3+1+⋯+n2n3+1)=lim⁡n→∞(16×n×(n+1)×(2n+1)(n+1)×(n2−n+1))=13\lim_{n \to \infty} \left ( \frac{1}{n^{3}+1 }+ \frac{4}{n^{3}+2 }+\dots+\frac{n^{2} }{n^{3}+n } \right )<\\ \lim_{n \to \infty} \left ( \frac{1}{n^{3}+1 }+ \frac{4}{n^{3}+1 }+\dots+\frac{n^{2} }{n^{3}+1 } \right )=\\ \lim_{n \to \infty} \left ( \frac{\frac{1}{6}\times n\times \left ( n+1 \right ) \times \left ( 2n+1 \right )}{\left ( n+1 \right )\times \left ( n^{2}-n+1\right) }\right )=\frac{1}{3}n→∞lim​(n3+11​+n3+24​+⋯+n3+nn2​)<n→∞lim​(n3+11​+n3+14​+⋯+n3+1n2​)=n→∞lim​((n+1)×(n2−n+1)61​×n×(n+1)×(2n+1)​)=31​

综上：

lim⁡n→∞(1n3+1+4n3+2+⋯+n2n3+n)=13\lim_{n \to \infty} \left ( \frac{1}{n^{3}+1 }+ \frac{4}{n^{3}+2 }+\dots+\frac{n^{2} }{n^{3}+n } \right )=\frac{1}{3} n→∞lim​(n3+11​+n3+24​+⋯+n3+nn2​)=31​
